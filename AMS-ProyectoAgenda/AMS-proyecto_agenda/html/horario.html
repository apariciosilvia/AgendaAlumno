ASIGNATURA: DISEÑO DE INTERFACES WEB

PROFESOR/A: CARMEN ARMIJO TORRES
HORAS A LA SEMANA: 6

CONTENIDO ASIGNATURA:

1. Principios de Diseño: Se estudian los fundamentos del diseño gráfico y la teoría del color, tipografía, y composición visual.

2. Usabilidad y Experiencia de Usuario (UX): Se enseña cómo crear interfaces que sean intuitivas y fáciles de usar, enfocándose en la satisfacción del usuario y la accesibilidad.

3. Herramientas de Diseño: Se utilizan herramientas como Adobe XD, Figma o Sketch para prototipar y diseñar interfaces.

4. Responsive Design: Se aprende a diseñar interfaces que se adapten a diferentes dispositivos y tamaños de pantalla, utilizando técnicas como grid y flexbox.

5. HTML y CSS: Se profundiza en la implementación de diseños utilizando HTML y CSS, incluyendo prácticas de codificación semántica y modular.

6. Prototipado y Pruebas: Se enseña a crear prototipos interactivos y a realizar pruebas de usabilidad para validar las interfaces diseñadas.

7. Tendencias Actuales: Se analizan las últimas tendencias en diseño web y se discuten casos de estudio para entender el impacto del diseño en el desarrollo de aplicaciones.


































ASIGNATURA: DESPLIGUE DE APLICACIONES WEB

PROFESOR/A: LETICIA RIAZA MORENO
HORAS A LA SEMANA: 5

CONTENIDO ASIGNATURA:

1. Entornos de Desarrollo y Producción: Se estudian las diferencias entre estos entornos y la importancia de una buena configuración para asegurar un despliegue exitoso.

2. Configuración de Servidores: Se abordan conceptos sobre servidores web (como Apache y Nginx), y se enseña cómo configurarlos para alojar aplicaciones.

3. Gestión de Bases de Datos: Se analiza la conexión y gestión de bases de datos, así como la migración de datos entre entornos.

4. Despliegue en la Nube: Se exploran plataformas de nube (como AWS, Google Cloud y Azure) y cómo utilizarlas para desplegar aplicaciones web de manera escalable y eficiente.

5. Contenedores y Virtualización: Se introduce el uso de Docker y otros sistemas de contenedorización para facilitar el despliegue y la gestión de aplicaciones.

6. Automatización de Despliegues: Se estudian herramientas y técnicas para automatizar procesos de despliegue, como CI/CD (Integración Continua/Despliegue Continuo).

7. Seguridad en el Despliegue: Se analizan las mejores prácticas para asegurar aplicaciones en producción, incluyendo configuraciones de seguridad y gestión de certificados SSL.

8. Monitoreo y Mantenimiento: Se enseña sobre la importancia del monitoreo de aplicaciones y cómo llevar a cabo un mantenimiento efectivo post-despliegue.


































ASIGNATURA: DESARROLLO WEB EN ENTORNO SERVIDOR

PROFESOR/A: JOSÉ ALBERTO LÓPEZ BALLESTEROS
HORAS A LA SEMANA: 8

CONTENIDO ASIGNATURA:

1. Introducción a la Programación del Lado del Servidor: Se presentan los principios básicos de la programación del lado del servidor, comparando con el desarrollo del lado del cliente.

2. Lenguajes de Programación: Se enseñan lenguajes como PHP, Node.js, Python (con frameworks como Django o Flask), o Ruby on Rails, enfocados en la creación de aplicaciones web dinámicas.

3. Frameworks y Librerías: Se exploran diferentes frameworks que facilitan el desarrollo, permitiendo una estructura más organizada y eficiente (como Express para Node.js o Laravel para PHP).

4. Gestión de Bases de Datos: Se estudian bases de datos relacionales (como MySQL y PostgreSQL) y no relacionales (como MongoDB), incluyendo operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y la integración con aplicaciones web.

5. APIs y Servicios Web: Se enseña a crear y consumir APIs RESTful, así como la integración con servicios web de terceros.

6. Autenticación y Autorización: Se abordan técnicas para gestionar la seguridad en las aplicaciones, incluyendo autenticación de usuarios y control de acceso.

7. Despliegue y Configuración del Servidor: Se discuten los procesos de despliegue de aplicaciones en servidores, incluyendo la configuración y optimización del entorno de servidor.

8. Manejo de Errores y Depuración: Se enseña a gestionar errores y a realizar depuración eficaz en aplicaciones del lado del servidor.

9. Pruebas y Mantenimiento: Se introducen técnicas de pruebas (unitarias y funcionales) y buenas prácticas para el mantenimiento de aplicaciones.


































ASIGNATURA: DESARROLLO WEB EN ENTORNO CLIENTE

PROFESOR/A: MIGUEL ANGEL CRUZADO VIVAS
HORAS A LA SEMANA: 6

CONTENIDO ASIGNATURA:

1. HTML y CSS: Se profundiza en la estructura de las páginas web utilizando HTML y el diseño y estilo con CSS, incluyendo técnicas avanzadas como Flexbox y Grid.

2. JavaScript: Se enseña el uso de JavaScript como el lenguaje de programación principal del lado del cliente, incluyendo conceptos básicos y avanzados, como manipulación del DOM, eventos y AJAX.

3. Frameworks y Librerías: Se exploran frameworks y librerías populares, como React, Vue.js o Angular, que facilitan la creación de aplicaciones web dinámicas y eficientes.

4. Manejo del Estado: Se estudian patrones y herramientas para gestionar el estado de las aplicaciones, especialmente en aplicaciones más complejas, como Redux o Vuex.

5. API y Consumo de Servicios Web: Se enseña a consumir APIs RESTful y a manejar datos de manera asíncrona utilizando tecnologías como Fetch y Axios.

6. Responsive Design y Accesibilidad: Se abordan prácticas para crear aplicaciones que funcionen en diversos dispositivos y que sean accesibles para todos los usuarios, cumpliendo con estándares de accesibilidad.

7. Herramientas de Desarrollo: Se introducen herramientas y entornos de desarrollo como Visual Studio Code, control de versiones con Git, y técnicas de depuración.

8. Pruebas en el Entorno Cliente: Se estudian métodos para realizar pruebas en las aplicaciones del lado del cliente, incluyendo pruebas unitarias y de integración con herramientas como Jest o Cypress.

9. Optimización de Rendimiento: Se abordan estrategias para optimizar el rendimiento de las aplicaciones web, incluyendo técnicas de carga diferida y minificación de recursos


































ASIGNATURA: INGLÉS

PROFESOR/A: ELENA MENCHERO GARRIDO
HORAS A LA SEMANA: 2

CONTENIDO ASIGNATURA:

1. Comprensión Lectora: Se trabajan textos técnicos, artículos y documentación relacionada con el desarrollo web, para familiarizarse con el vocabulario y las estructuras del inglés técnico.

2. Expresión Escrita: Se practica la redacción de correos electrónicos, informes y documentación técnica, fomentando el uso adecuado del lenguaje y la claridad en la comunicación.

3. Expresión Oral: Se desarrollan habilidades de conversación a través de presentaciones y discusiones sobre temas relacionados con la programación y el desarrollo web, para mejorar la fluidez y la confianza al hablar.

4. Vocabulario Técnico: Se estudian términos específicos del ámbito de la informática y el desarrollo web, incluyendo lenguajes de programación, frameworks y metodologías de trabajo.

5. Escucha Activa: Se realizan actividades de comprensión auditiva mediante podcasts, conferencias y videos en inglés relacionados con el desarrollo y las tendencias tecnológicas.

6. Gramática y Estructuras: Se refuerzan las reglas gramaticales y las estructuras del inglés, aplicándolas en contextos técnicos y profesionales.

7. Cultura Profesional: Se aborda la cultura empresarial y profesional en entornos angloparlantes, incluyendo prácticas de trabajo en equipo y habilidades interpersonales.

8. Preparación para Certificaciones: En algunos casos, se pueden incluir preparativos para exámenes de certificación en inglés, como el B1 o B2, que son valorados en el ámbito laboral.


































ASIGNATURA: EMPRESA E INICIATIVA EMPRENDEDORA

PROFESOR/A: M.PILAR ÁLVAREZ SOUSA
HORAS A LA SEMANA: 3

CONTENIDO ASIGNATURA:

1. Conceptos Básicos de Empresa: Se introducen los fundamentos sobre qué es una empresa, tipos de empresas, estructura organizativa y funciones empresariales.

2. Emprendimiento: Se analizan las características del emprendedor y el proceso de emprendimiento, así como las habilidades necesarias para iniciar y gestionar un negocio.

3. Plan de Negocio: Se enseña a desarrollar un plan de negocio, que incluye la definición de la idea, análisis de mercado, planificación financiera y estrategias de marketing.

4. Innovación y Creatividad: Se fomenta la creatividad e innovación como motores del emprendimiento, analizando cómo generar ideas y convertirlas en proyectos viables.

5. Financiación de Proyectos: Se estudian las diferentes fuentes de financiación disponibles para emprendedores, incluyendo capital propio, préstamos, subvenciones y financiación a través de inversores.

6. Aspectos Legales: Se abordan los aspectos legales y administrativos relacionados con la creación de una empresa, incluyendo tipos de sociedades, derechos y obligaciones.

7. Marketing y Ventas: Se analizan las estrategias de marketing y ventas, así como la importancia de conocer el mercado objetivo y cómo llegar a los clientes.

8. Gestión de Recursos: Se estudia la gestión eficaz de recursos humanos, materiales y financieros para asegurar el funcionamiento óptimo de la empresa.

9. Casos Prácticos y Simulaciones: Se utilizan casos de estudio y simulaciones para aplicar los conceptos aprendidos y entender los retos que enfrentan los emprendedores en la realidad.
